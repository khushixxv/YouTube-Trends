Project Title: YouTube Trends

Authors: Khushi Patel & Cynthia Chen

Desciption: Our project, YouTube Trends, aims to look at a dataset containing the top 200 YouTube videos daily in 11 different countries. By utilizing our data visualizations, both consumers and creators will be able to use this data to have a deeper understanding of how trends grow and spread on YouTube, and how to manipulate these trends.

Installation Procedures:
Ensure Flask is installed to run project locally.
Numpy, Pandas, Plotly are required for running the python files. 

To Run:
$export FLASK_APP=app
$export FLASK_ENV=development
$flask run
